# Enhanced JARVIS Voice Assistant

🤖 **Advanced Voice-Controlled Personal Assistant** inspired by Iron Man's JARVIS AI

## Quick Start

1. Install dependencies: `pip install -r requirements.txt`
2. Run enhanced version: `python enhanced_main.py`
3. Say "JARVIS" followed by your command
4. Open `index.html` for web interface

## Features

✅ **25+ Song Music Library** with categories and playlists  
✅ **Advanced Voice Recognition** with multiple engines  
✅ **Website Automation** (Google, YouTube, Instagram, etc.)  
✅ **System Control** (Apps, shutdown, volume control)  
✅ **Smart Conversations** with Wikipedia integration  
✅ **Modern Web UI** with real-time visualizations  

## Voice Commands Examples

- "JARVIS, play Kesariya"
- "JARVIS, open YouTube"
- "JARVIS, what time is it?"
- "JARVIS, open Calculator"
- "JARVIS, tell me about AI"

See `SETUP_GUIDE.md` for complete documentation.

## Files

- `enhanced_main.py` - Main enhanced application
- `enhanced_music_library.py` - 25+ song database
- `main.py` - Original version
- `index.html` - Web interface
- `requirements.txt` - Dependencies

## Requirements

- Python 3.7+
- Microphone
- Internet connection
- pip packages (see requirements.txt)
